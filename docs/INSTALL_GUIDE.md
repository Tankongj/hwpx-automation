# HWPX Automation 설치 가이드

> 지인에게 전달할 수 있는 한 장짜리 설치 안내서입니다.
> 인쇄해서 드리거나, 이 페이지 링크만 공유하셔도 OK.

---

## 🎁 3단계면 끝

### ① 설치 파일 다운로드

👉 https://github.com/Tankongj/hwpx-automation/releases/latest

위 주소 들어가셔서 페이지 아래쪽 **Assets** 섹션의 **`HwpxAutomation-Setup-v0.xx.x.exe`** 파일을 클릭해 받으세요.

```
Assets
  📦 HwpxAutomation-Setup-v0.17.5.exe     ← 이걸 받으세요
  📦 full.zip                              (무시)
  📄 manifest.json                         (무시)
  📄 Source code (zip)                     (무시)
  📄 Source code (tar.gz)                  (무시)
```

### ② 받은 `.exe` 파일을 더블클릭

한국어 설치 마법사가 뜹니다:

```
  [HWPX Automation 설치]

  ──────────────────────────────
  HWPX Automation 설치 마법사에
  오신 것을 환영합니다

  한글 문서 자동화 데스크톱 앱을
  컴퓨터에 설치합니다.

  계속하려면 "다음" 을 누르세요.
  ──────────────────────────────

           [다음 ►]
```

**"다음"** 을 2~3번 누르면 **"설치"** 버튼이 나옵니다. 눌러주세요.

### ③ ⚠ 파란색 "Windows에서 PC를 보호했습니다" 경고가 뜨면

당황하지 마세요. 이건 정품 인증서가 없는 신규 앱에 대한 Windows 의 일반적 경고입니다. **바이러스 아닙니다**.

1️⃣ **"추가 정보"** 를 클릭하세요 (작은 파란 글씨)

```
  ┌──────────────────────────────────┐
  │ Windows에서 PC를 보호했습니다   │
  │                                  │
  │ Microsoft Defender SmartScreen이  │
  │ 인식할 수 없는 앱의 시작을      │
  │ 차단했습니다.                   │
  │                                  │
  │ [추가 정보] ← 여기 클릭!        │
  │                                  │
  │        [실행하지 않음]           │
  └──────────────────────────────────┘
```

2️⃣ **"실행"** 버튼이 나타나면 클릭

```
  ┌──────────────────────────────────┐
  │ 게시자: 알 수 없는 게시자       │
  │ 앱:     HwpxAutomation-Setup-... │
  │                                  │
  │ [실행] ← 여기!                  │
  │                                  │
  │        [실행하지 않음]           │
  └──────────────────────────────────┘
```

## 🎉 설치 완료 후

- **바탕화면** 에 **HWPX Automation** 아이콘 생성됨 → 더블클릭으로 실행
- **시작 메뉴** 에서도 "HWPX Automation" 으로 찾을 수 있음
- 다음번부턴 경고 없이 바로 실행됨

## 🔄 새 버전이 나오면?

재설치 안 해도 됩니다. 앱이 **자동으로** 새 버전을 찾아서:
1. "업데이트 받으시겠습니까?" 창 표시
2. **"예"** 클릭 → 자동 다운로드 + 재시작
3. 끝

사용자 설정·API 키 등은 그대로 유지됩니다.

## 🆘 설치 중 문제가 생기면

### "이 앱은 PC에서 실행되지 않습니다" 메시지
- 64-bit Windows 10 이상 필요. Windows 7, 32-bit 환경은 지원 안 됨.

### "다른 사용자가 이미 설치하고 있습니다" 메시지
- 작업 관리자 (Ctrl+Shift+Esc) 에서 `HwpxAutomation-Setup` 프로세스 종료 후 재시도

### 다운로드 속도가 너무 느림
- 회사 방화벽에서 GitHub 다운로드 제한 가능성. 개인 PC 또는 스마트폰 테더링으로 재시도.

### 그 외 문제
- GitHub Issues: https://github.com/Tankongj/hwpx-automation/issues
- 또는 전달해주신 분께 문의

## 📱 제거 방법

앱이 마음에 안 들거나 지우고 싶으면:

1. **설정** 열기 (Windows 키 + I)
2. **앱** → **설치된 앱**
3. **HWPX Automation** 찾아서 **제거**

사용자 설정 (API 키, 템플릿) 은 보존되므로 나중에 재설치 시 그대로 복원됩니다. 완전 삭제하려면 `%APPDATA%\HwpxAutomation` 폴더도 수동 삭제해주세요.
